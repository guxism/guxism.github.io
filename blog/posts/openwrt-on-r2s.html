R2S 上运行 OpenWrt

Er-X 虽然牛逼，但是硬路由性能始终不行，OpenClash + DNSmaql 差不多占了 90% CPU, 导致上网速度变慢，我注意到这是我将 3 台设备接入 Er-X 之后的惨状。

二话不说，直接把 r2s 替补上。

这圈子内流传着各种自定义固件，但是我不相信这些东西，所以直接刷 官方 Openwrt 的固件： https://openwrt.org/toh/friendlyarm/nanopi_r2s

安装第二个镜像：

理由是对 flash 友好

用 balenaEtcher 刷到 SDCard. 通电，启动, 默认 192.168.1.1

刷 OpenClash

先安装这些依赖：

#iptables
opkg update
opkg install coreutils-nohup bash iptables dnsmasq-full curl ca-certificates ipset ip-full iptables-mod-tproxy iptables-mod-extra libcap libcap-bin ruby ruby-yaml kmod-tun kmod-inet-diag unzip luci-compat luci luci-base

#nftables
opkg install coreutils-nohup bash dnsmasq-full curl ca-certificates ipset ip-full libcap libcap-bin ruby ruby-yaml kmod-tun kmod-inet-diag unzip kmod-nft-tproxy luci-compat luci luci-base

这篇文章讲到如何扩展根目录空间： https://blog.niekun.net/archives/2321.html

我需要操作的是 squashfs

哦，官网也有说明： https://openwrt.org/docs/guide-user/installation/installation_methods/sd_card#expanding_the_filesystem

但这说明是错的，squashFS 一旦写入，就不能再更改，具体原因不明，以后再理解

唯一可行的做法是，刚好刷出来的 SD-CARD, 直接分区，不需要 fsck.f2fs 或者 resize.f2fs
