HDL

芯片 And 的实现:

CHIP And {
    IN a, b;
    OUT out;

    PARTS:
    Nand(a=a, b=b, out=c);
    Not(in=c, out=out);
    // Put your code here:
}

输入是 a, b, 输出 out

组成:

  • Nand gate, a = a, IN a 为输入, 相当于将 IN a 连接到 Nand 的 a 引脚, Nand 的 out 引脚被标注为 c
  • c 接入 Not, 输出到 Out out

Hack Assembly

寄存器

A

地址. M, 和 jump 都使用这个寄存器存储的地址

M

凡是用到 M, 都用到地址 A 指定位置

D

Data

A-instruction(Address)

@value

将 A 寄存器的值设置为 value

C-instruction(Computation)

dest=comp;jump

jump

Logic:

if Memory[3] = 5 then goto 100
else goto 200

实现:

@3
D=M     ; 从 M[3] 中读取数据放在 D
@5      ; 将 A 设置为 5
D=D-A   ; D -= 5
@100    ; A = 100
D;JEQ   ; 如果 D == 0, 跳到 A(100)
@200    ; A = 200
0;JMP   ; 跳到 A(200)

特殊地址

  • 16384: SCREEN, 屏幕
  • 24576: KBD, 键盘

示例: 按键的时候, 屏幕变黑

(INITIATION)        ; 初始化
    @24576          ; A = 24576 指向键盘
    D=A             ; D = A
    @SCROPENCLOSE   ; 声明 SCROPENCLOSE 变量并指定将这个值赋予 A. 自定义变量从 16(0x0010) 开始
    M=D             ; M[16] = 24576
    @color          ; A = color = 17
    M=0             ; M[17] = 0
    @addr           ; A = addr = 18

(LISTEN)            ; 监听键盘
    @KBD            ; A = 键盘
    D=M             ; 取键盘值
    @CLEAR          ; A = CLEAR Subroutine
    D;JEQ           ; 如果键盘值为 0, 跳到 CLEAR
    @PAINTBLACK0    ; 否则, 将 A 设置为 PAINTBLACK0, 顾名思义, 是将屏幕涂黑的意思
    0;JMP           ; 无条件跳转

(CLEAR)             ; 将屏幕清空
    @color          ; 取 color 值
    D=M         
    @PAINTWHITE0
    D;JGT           ; 如果 D 大于 0, 将屏幕涂成白色
    @LISTEN         ; 
    0;JGT           ; 否则, 继续监听
