libuv

libuv 和 libev 的区别: https://gist.github.com/andreybolonin/2413da76f088e2c5ab04df53f07659ea

  1. libev 没有异步 DNS 解析(asynchronous DNS resolution), 意味着 libev 会堵塞在域名解析上; libuv 有
  2. libev 更像是个人项目
  3. libev 只是 system I/O 复用的简单封装, libuv 更抽象, 有线程池
