#

链表的数学定义

数组的索引为 x, 函数为 f(x)

{1, 2, 3, 4} 表示第一个节点, 地址 0, 指向地址 1, 这就是一个直链

{1, 2, 3, 4, 2} 地址 3 的节点指向地址 2

                      +-------------------+
                      v                   |
+---+     +---+     +---+     +---+     +---+
| 0 | --> | 1 | --> | 2 | --> | 3 | --> | 4 |
+---+     +---+     +---+     +---+     +---+

正式定义:

设 S 为有限集合, f 为任意函数, x0 是 S 的任意元素, 对于任意 i > 0, 设 xih=f(xi−1), μ 为 环入口, λ 为环的长度, 有 xμ=xλ+μ, 环状检测的任务是找到 μ 和 λ

#

Floyd’s tortoise and hare

快慢指针, 快的叫 hare, 慢的叫 tortoise

问题的关键: 如果链表存在环, 则指针永远不可能达到 NULL, 对于任意 i≥μ, 有 xi=xi+kλ, 其中 k 为常数, λ 是环的周长

推导:

  • 假设龟兔相遇的时候 tortois 跑了 kt 圈, hare kh 圈, 因为 hare 的速度是 tortoise 的两倍, 所以路程也是 2: 1, 所以有 2(μ+ktλ+t)=μ+khλ+t
  • 得出: μ=(kh−ht)λ−t
  • 这两者在距离环入口 t 处相遇, 怎么做到让它们在入口相遇? 让 tortoise 从原点开始往前走, 同时 hare 以相同速度转圈, tortoise 走 μ 步之后, hare 走了 (kh−ht)λ−t 步
  • 注意到 hare 是从偏移量为 +t 的位置开始走的, 这意味着, 它们相遇的时候, hare 还差 −t 才满足 kλ 即整数圈, 意味着 hare 刚好处于环的入口.
#

#

再次相遇之后, 让 tortoise 保持位置不变, hare 往前, 再次相遇的时候 hare 的路径即是环的周长.

#

还有两种算法: Brent’s algorithm 和 Gosper’s algorithm, 是对 Floyd’s algorithm 的优化, 为节省脑力, 不做了解.
